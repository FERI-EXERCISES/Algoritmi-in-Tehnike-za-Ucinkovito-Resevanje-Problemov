# Vaja 2: Pomorska flota (2023/2024)

Implementirajte rešitev podane naloge s podatkovno strukturo za disjunktne množice. Program naj prebere vhodne podatke iz datoteke, rezultat pa izpiše v konzolo. Oddajte ZIP datoteko izvorne kode (brez direktorijev bin, obj, Debug, Release, itd...).

Primer zagona programa: ./vaja2.exe testni_primer1.txt

Dovoljeni programski jeziki: C++, Python.
(C++ implementacije bodo prevedene in preizkušene v Visual Studiu 2022/JetBrains CLion, Python implementacije pa pognane s Python različico >= 3.8)

Vrednost vaje: 10 točk.

Rok za oddajo: 29. 3. 2024 ob 6h zjutraj, rok za zagovor: 29. 3. 2024.

Pravilni rezultati testnih primerov:
- testni_primer1: rešitev na voljo v priponki
- testni_primer2: rešitev na voljo v priponki

Opomba:
- Po želji lahko uporabljate objektno ali funkcijsko programiranje.

Pomoč:
- V primeru, da pride do stack overflow-a pri najtežjem testnem primeru, poizkusite zvečati velikost sklada. V Visual Studiu se to naredi: Properties > Configuration Properties > Linker > System > Stack Reserve size, nastavite vrednost npr. 12194304.
